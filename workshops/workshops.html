Calendar of weekly workshops at Spartan hackers!

Pull this from the workshop resource:

  • GET workshops: Get list of workshop events use in calendar widget
  • POST/PUT workshops: Add new workshop

Click an event, get a page from resource:

  • GET workshops/{id}: Get info about a workshop
  • GET workshops/{id}/attendees: Get attendees
  • POST workshops/{id}: Update info about a workshop
  • POST workshops/{id}/attendee/{id}: Add user to attendees list IF event is going on.
  • DELETE workshops/{id}: Remove workshop
  • DELETE workshops/{id}/attendees/{id}: Remove user from event (Yourself or admin only)
