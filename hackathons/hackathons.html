List of hackathons Spartan hackers is arranging teams and travel for!

Pull this from the hackathons resource:

  • GET hackathons: Get list of hackathon events use in calendar widget
  • POST/PUT hackathons: Add new hackathon

Click an event, get a page from resource:

  • GET hackathons/{id}: Get info about a hackathon
  • GET hackathons/{id}/attendees: Get attendees
  • POST hackathons/{id}: Update info about a hackathon
  • POST hackathons/{id}/attendee/{id}: Add user to attendees list IF event is going on.
  • DELETE hackathons/{id}: Remove hackathon
  • DELETE hackathons/{id}/attendees/{id}: Remove user from event (Yourself or admin only)
